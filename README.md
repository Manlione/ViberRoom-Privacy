# VibeRoom Legal Pages

Questa repository contiene la pagina **Termini & Condizioni + Privacy Policy** dell'app VibeRoom.

## Pubblicazione tramite GitHub Pages

1. Vai su **Settings** → **Pages**  
2. Source: seleziona **Main Branch**  
3. Directory: **Root**  
4. Salva

Il sito sarà disponibile a un link simile a:

`https://MANLIONE.github.io/viberoom-legal/`

Usa questo link nell'app per mostrare i documenti legali.
